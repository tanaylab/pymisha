# API Reference

Complete API reference for pymisha. Functions are organized by category.

| Section | Description |
|---------|-------------|
| [Database](database.md) | Initialize, configure, and manage genomic databases |
| [Datasets](datasets.md) | Load, save, and manage named datasets |
| [Tracks](tracks.md) | Create, import, modify, and manage genomic tracks |
| [Virtual Tracks](vtracks.md) | Define computed views over existing tracks |
| [Intervals](intervals.md) | Create, manipulate, and query genomic intervals |
| [Data Operations](data-operations.md) | Extract, summarize, and analyze track data |
| [Liftover](liftover.md) | Convert coordinates between genome assemblies |
| [Sequence Analysis](sequence.md) | Extract and analyze DNA sequences |
| [Genome Synthesis](synthesis.md) | Train generative models and sample synthetic genomes |
